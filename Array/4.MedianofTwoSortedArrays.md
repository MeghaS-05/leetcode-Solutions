class Solution(object):
    def findMedianSortedArrays(self, nums1, nums2):
        merged = sorted(nums1+nums2)
        n = len(merged)
        if n%2==1:
            return float(merged[n//2])
        else:
            return(merged[n//2-1]+merged[n//2])/2.0
            


# Explanation :
Choose partition in nums1
        ↓
Find partition in nums2
        ↓
Check if partition is valid
        ↓
      Yes? ───► Compute median
        │
        No
        ↓
Move partition left/right (Binary Search)
        ↓
Repeat

Time Complexity is O(log(min(m,n))) because we have smaller arrays and binary search is useful for smaller arrays
Space Complexity will be O(1) since we have few variable used in the logic.
