# Code:
class Solution(object):
    def twoSum(self, nums, target):
        n = len(nums)
        #main logic
        for i in range(n):
            for j in range(i+1, n):
                if nums[i] + nums[j] == target:
                    return [i,j]




# Explanation :
The idea is to check every possible pair of numbers in the array.

Outer loop (i):
Picks one element at a time.
Think of it as fixing the first number of the pair.

Inner loop (j):
Starts from i + 1 so we don't compare an element with itself or repeat pairs.
Checks every element after i.
Check the sum
If nums[i] + nums[j] == target, we've found the required pair.
Return their indices [i, j].

# Example :
Input:
nums = [2, 7, 11, 15]
target = 9
here 2+7 is 9 and rest combination won't match with the target so our output would be [0,1].


Time Complexity will be O(n^2) since both outer and inner loop runs n times.
Space Complexity will be O(1) since only few variable like n,i,j is used.


Since 2 + 7 = 9, return:

[0, 1]
