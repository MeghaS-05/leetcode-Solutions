# Code :
class Solution:
    def maxArea(self, height):
        left = 0
        right = len(height) - 1
        max_area = 0
        while left < right:
            width = right - left
            area = width * min(height[left], height[right])
            max_area = max(max_area, area)
            if height[left] < height[right]:
                left += 1
            else:
                right -= 1
        return max_area

        

# Explanation :
Start with two pointers at the beginning and end of the array, giving the maximum possible width. At each step, calculate the container's area as:

area = (right - left) * min(height[left], height[right])

Update the maximum area found so far. Then move the pointer pointing to the shorter line, because the shorter line limits the container's height. Moving the taller line only decreases the width without increasing the limiting height, so it cannot produce a larger area. Repeat until the pointers meet.

Time complexity is O(n) since each pointer moves inward at most n times.
Space complexity is O(1) becuz only a few variables are used.
